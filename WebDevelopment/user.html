Projects

üîå Smart Shopping Cart

This project was designed to make shopping easier and smarter. Every time an item is placed into the cart, it is scanned using RFID, and the item details plus the total price are displayed on an interactive screen. The price updates automatically as more items are added or removed. After shopping, the user receives a prompt to pay via M-Pesa through their phone. If payment is not detected within a set time, a buzzer sounds an alarm, which can only be stopped by a staff member entering a PIN. The system also uses ESP8266 NodeMCU and CallMeBot to send an automatic WhatsApp message to the shopper, enhancing communication and security.
